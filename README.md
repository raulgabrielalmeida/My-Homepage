# Custom Browser Homepage

A lightweight and customizable browser homepage with integrated Google search.

## Description

This project provides a minimal and fast homepage designed to replace the default start page in a web browser. It focuses on simplicity and efficiency by offering a clean interface with direct access to Google search.

The homepage is ready to use via a hosted link and does not require installation. It is primarily configured for Firefox, with support for both homepage and new tab customization.

## Getting Started

No installation is required to use this project.

Simply access the hosted version:

`https://raulgabrielalmeida.github.io/My-Homepage/`

---

### Optional (for customization)

If you want to modify or customize the homepage:

1. Clone the repository:

```
git clone https://github.com/raulgabrielalmeida/My-Homepage.git
```

2. Edit the project files (`index.html`, `style.css`, etc.)

3. Open `index.html` in your browser or deploy your own version

---

## Usage

### Option 1 — Set as Homepage (Ctrl + N)

1. Open Firefox
2. Click on the menu (≡)
3. Go to **Settings**
4. Navigate to **Home**
5. In "Homepage and new windows", select **Custom URLs**
6. Paste your page URL:
   `https://raulgabrielalmeida.github.io/My-Homepage/index.html`

---

### Option 2 — Set as New Tab (Ctrl + T)

This requires an extension.

1. Install the extension:
   New Tab Override
   `https://addons.mozilla.org/pt-BR/firefox/addon/new-tab-override/`

2. After installing:

   * Open a new tab (Ctrl + T)
   * Select **Custom URL**
   * Paste your page URL:
     `https://raulgabrielalmeida.github.io/My-Homepage/index.html`
   * Apply changes (adjust shortcut if needed)

---

## Help

Common issues:

* Page not loading:

  * Ensure the URL is correct
  * Prefer using the hosted version instead of a local file

* New tab not changing:

  * Confirm the extension is installed and enabled
  * Verify that the custom URL is correctly configured
